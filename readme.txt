== Installation ==

= The easy way: =

1. Go to the Plugins Menu in WordPress
1. Search for "Tax Calculator"
1. Click 'Install'
1. Activate the Plugin


Please check DEMO on this Link http://www.github.com/ .

= Manual Installation =

1. Download the plugin file from this page and unzip the contents
1. Upload the `tax-calculator` folder to the `/wp-content/plugins/` directory
1. Activate the `tax-calculator` plugin through the 'Plugins' menu in WordPress

= Once Activated =

1. Place the `[tax-calculator]` shortcode in a Page or Post.
